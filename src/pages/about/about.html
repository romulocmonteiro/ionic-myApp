Sobre este Aplicativo

Este aplicativo está sendo desenvolvido utilizando as tecnologias Node.js, Angular.js, Ionic 3 e Cordova para geração de um aplicativo híbrido para celulares Androis e iOS.

Neste aplicativo farei testes de criação telas em Slides, Tabs e Menus fazendo a transição dinâmica de dados entre as diferentes telas do aplicativo.

A prmeira funcionalidade construída é o TMDB, um feed de filmes sendo exibidos em cinemas do Brasil, utilizando como base os métodos da API do site TMDB. Nesta funcionalidade, serão apresentadas 3 telas:

  • Lista de filmes em cartaz (sem paginação)
  • Tela de detalhes de um filme selecionado
  • Tela de detalhes de um filme semelhante, selecionado na tela anterior
